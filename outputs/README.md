# Generated Outputs

This directory contains MRD files and other outputs generated by the tool.

## Directory Structure

- `config_test_output/`: MRDs generated during configuration testing
- `high_perf_output/`: High-performance configuration outputs  
- `ras_output/`: RAS (Reliability, Availability, Serviceability) specific outputs
- `test_output/`: General testing outputs
- `underscore_test_output/`: Outputs from underscore formatting tests
- `web_output/`: Files generated through web interface

## Usage

Generated MRD files in these directories can be:
- Validated using `python3 mrd_tool.py validate -f <file>`
- Used as examples for integration
- Compared for regression testing

## Cleanup

Old outputs can be removed periodically. Keep important reference outputs for comparison.
